# Codec Technologies Internship Projects

Data Analytics internship at Codec Technologies. Two projects completed using Python, Pandas, and data visualization.

### 1. Sales Forecast Analysis
Time-series sales forecasting using Prophet with Indian holidays. Validated with MAPE on 30-day holdout. Identified seasonal trends and predicted future sales to support inventory planning.

**File:** [Sales forecast - Colab.pdf](./Sales%20forecast%20-%20Colab.pdf)

### 2. Social Media Engagement Dashboard
Analyzed Instagram engagement metrics to find content patterns. Photo 1 reached 6,563 users vs ~400 avg for others. Recommended posting 7-9 PM IST and testing Reels format to improve reach.

**File:** [SOCIAL MEDIA ENGAGEMENT BOARD.pdf](./SOCIAL%20MEDIA%20ENGAGEMENT%20BOARD%20.pdf)

### Tools Used
Python, Pandas, Prophet, Matplotlib, Google Colab

Built for Codec Technologies 1-Month Data Analytics Internship.
